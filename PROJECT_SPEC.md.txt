PROJECT SPECIFICATION
Project Name
Finance & Tutor Management Platform

PROJECT GOAL
Создать современное SaaS-веб-приложение, объединяющее:

Финансовый трекер
CRM для репетиторства
Календарь занятий
Систему домашних заданий
Кабинет ученика
Администрирование пользователей
Проект должен быть production-ready и готов к реальному использованию.

CRITICAL REQUIREMENTS
Предыдущая версия проекта столкнулась с проблемами на этапе деплоя.
Поэтому новый проект должен быть спроектирован с учетом:
простого деплоя
минимального количества сервисов
бесплатного размещения
минимальных расходов
высокой надежности
Если существует несколько технически корректных решений, всегда выбирать:
более простой вариант
более дешевый вариант
более легко поддерживаемый вариант
